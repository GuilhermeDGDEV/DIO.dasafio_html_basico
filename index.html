Estrutura Básica do HTML

(Voltar)

A estrutura básica que é formada uma página em HTML é formado pela estrutura conforme abaixo:
<html>
  <head>
  </head>
  <body>
  </body>
</html>
A tag que engloba todos os compoentes é a própria tag html, onde ela possue duas tags dentro dela, que são a head (onde ficarão as configurações da página) e a tag body (que é onde ficará o conteúdo do site)

Falando sobre Tags

(Voltar)

Além das tags mais básicas para construir a estrutura básica do HTML, também existem outras tags para contrução do mesmo, como por exemplo div, article, menu, header e assim por diante, levando em conta também as tags que não possuem tags de fechamento, como exemplos abaixo:
<input />
<img />

Atributos das Tags

(Voltar)

As tags possuem também um tipo de configuração que são os atributos. Os atributos que todas as tags possuem em comum são os atributos id (que é usado normalmente como uma forma de identificar que aquela tag como elemento único) e a class (que normalmente é usado para marcar como estilo, usando CSS). Exemplo:
<div id="artigo_principal" class="subtitulo texto-negrito"></div>
Mas também existem tags que possuem atributos especificos que funcionam apenas neles. Exemplo:
<input type="text" />

Textos

(Voltar)

Para fazer a formatação de texto no html, existem algumas tags como <p> para parágrafo, <string> para texto negrito, <i> para itálico e <blockquote> para citação, basta apenas por o conteúdo dentro dessas tags para que o texto fique formatado.

Listas Ordenadas e Não Ordenadas

(Voltar)

Para criar listas, são usadas as tags ul (lista não ordenada) e ol (lista ordenada), e usando a tag li adentro de ambos para criar os itens da lista. Exemplo:
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

<ol>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>

Links

(Voltar)

A tag responsável por fazer a navegação tanto para outro site ou para um conteúdo no mesmo site é a tag <a>. Nessa tag é obrigatório o uso do atributos href, que é onde terá o destino para onde o link irá levar.

Exemplos:

Link para a mesma guia:
<a href="google.com"></a>

Link para outra guia:
<a href="google.com" target="_blank"></a>

Link para o conteúdo do mesmo site:
<a href="#id"></a>
